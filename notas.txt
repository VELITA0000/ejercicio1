Hash vs Encriptado
En el hash tenemos un factor que modifica el valor y ya no se conoce, si se quiere validar hasheamos el input y comparamos el hash
En el encriptado se transforma el valor y tenemos un factor para transformarlo y poder leerlo

Con Hash se guarda con un token los datos de un usuario en la base de datos pero lo vuelve mas lento, solo hay un login 
JWT encriptamos con el token tenemos el user id pero estamos limitados al control de ciertas cosas del usuario como el login
